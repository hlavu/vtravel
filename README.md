# Vtravel

Vtravel is a travel website for promoting trips to Vietnam. 

## Clone repo

```
git clone https://github.com/hlavu/vtravel.git
```

## Host locally

```
npm install
npm run build

open live server
```

## Built with
- HTML
- JS
- Tailwindcss
